Instructions
------------

The directories contain several short coding problems.

Each file has instructions on the top.

You are asked to do as many of the problems as you wish.

The code should build on any recent Linux distribution.

Please feel free to ask questions.

Directory: debug
----------------

Contains small, simple programs - which have bugs.

You are asked to debug and fix the bugs.

Directory: coding
-----------------

Contains small programs, parts of which are missing.

You are asked to implement the missing parts.
